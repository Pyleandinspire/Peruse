## <a name="chinese"></a> 中文

### 📱 软件简介

**长物Peruse** 是一款优雅的长期消费记录应用，帮助用户追踪物品的使用价值和消费趋势。通过记录购买物品的价格和日期，自动计算平均每日成本，让用户更理性地看待消费。

### ✨ 核心功能

- **物品管理**：添加、编辑、删除物品
- **价值计算**：自动计算每个物品的平均每日成本
- **统计分析**：显示总价值和平均每日开销
- **数据导出**：支持将数据导出为文本文件
- **数据导入**：支持从文本文件导入数据
- **批量操作**：支持批量选择和删除物品
- **使用说明**：内置详细的使用指南

### 📦 安装说明

#### 系统要求

- Windows 10/11
- Android 6.0+
- Web浏览器（Chrome、Firefox、Safari等）
- mac和IOS设）

#### 安装方法

**Windows**

1. 下载压缩包
2. 无需安装，解压即用

**Android**

1. 下载最新的APK文件
2. 在设备上允许安装来自未知来源的应用
3. 点击APK文件进行安装

### 🚀 使用指南

在app内点击顶上的“长物Peruse”，即可查看使用说明。

#### 添加物品

1. 点击右下角的「+」按钮
2. 输入物品名称、购买价格和购买日期
3. 点击「添加」按钮保存

#### 查看统计信息

- 应用顶部显示总价值和平均每日开销
- 每个物品卡片显示购买价格、购买日期和使用天数

#### 编辑物品

1. 点击物品卡片上的编辑图标
2. 修改物品信息
3. 点击「保存」按钮

#### 删除物品

1. 点击物品卡片上的删除图标
2. 确认删除操作

#### 批量操作

1. 点击顶部的「批量删除」按钮
2. 选择要删除的物品
3. 点击「删除」按钮确认删除

#### 导出数据

1. 点击顶部的「导出」按钮
2. 选择保存位置
3. 数据将保存为文本文件

#### 导入数据

1. 点击顶部的「导入」按钮
2. 选择包含数据的文本文件
3. 数据将被导入到应用中

### ⚙️ 配置选项

- **主题**：应用使用浅色主题，符合现代设计美学
- **数据存储**：所有数据保存在本地，确保隐私安全
- **文件格式**：导出/导入使用简单的文本格式（名称|价格|日期）

### 🔧 故障排除

#### 常见问题

**无法导出文件**

- 检查是否有足够的存储空间
- 确保应用有文件访问权限
- 尝试使用不同的保存位置

**导入失败**

- 检查文件格式是否正确
- 确保文件编码为UTF-8
- 验证文件内容格式是否为「名称|价格|日期」

**应用崩溃**

- 尝试重启应用
- 检查是否有足够的内存
- 如果问题持续，尝试重新安装应用

#### 错误提示

- **「没有数据可导出」**：应用中没有添加任何物品
- **「导入失败」**：文件格式错误或文件损坏
- **「删除失败」**：尝试删除不存在的物品

## <a name="english"></a> English

### 📱 Software Introduction

**Peruse** is an elegant long-term consumption tracking application that helps users monitor the value and consumption trends of their possessions. By recording the price and date of purchased items, it automatically calculates the average daily cost, allowing users to view consumption more rationally.

### ✨ Core Features

- **Item Management**：Add, edit, and delete items
- **Value Calculation**：Automatically calculate average daily cost for each item
- **Statistical Analysis**：Display total value and average daily expense
- **Data Export**：Support exporting data to text files
- **Data Import**：Support importing data from text files
- **Batch Operations**：Support batch selection and deletion of items
- **Usage Guide**：Built-in detailed usage instructions

### 📦 Installation Instructions

#### System Requirements

- Windows 10/11
- Android 6.0+
- Web browser (Chrome, Firefox, Safari, etc.)
- Mac and iOS devices

#### Installation Methods

**Windows**

1. Download the compressed package
2. No installation required, just unzip and use

**Android**

1. Download the latest APK file
2. Allow installation from unknown sources on your device
3. Tap the APK file to install

### 🚀 Usage Guide

Click on "长物Peruse" at the top of the app to view the usage instructions.

#### Adding Items

1. Tap the "+" button in the bottom right corner
2. Enter item name, purchase price, and purchase date
3. Tap the "Add" button to save

#### Viewing Statistics

- The app displays total value and average daily expense at the top
- Each item card shows purchase price, purchase date, and days used

#### Editing Items

1. Tap the edit icon on the item card
2. Modify the item information
3. Tap the "Save" button

#### Deleting Items

1. Tap the delete icon on the item card
2. Confirm the deletion operation

#### Batch Operations

1. Tap the "Batch Delete" button at the top
2. Select the items to delete
3. Tap the "Delete" button to confirm deletion

#### Exporting Data

1. Tap the "Export" button at the top
2. Select a save location
3. Data will be saved as a text file

#### Importing Data

1. Tap the "Import" button at the top
2. Select a text file containing data
3. Data will be imported into the app

### ⚙️ Configuration Options

- **Theme**：The app uses a light theme, consistent with modern design aesthetics
- **Data Storage**：All data is stored locally, ensuring privacy and security
- **File Format**：Export/import uses a simple text format (name|price|date)

### 🔧 Troubleshooting

#### Common Issues

**Cannot Export File**

- Check if there is enough storage space
- Ensure the app has file access permissions
- Try using a different save location

**Import Failure**

- Check if the file format is correct
- Ensure the file encoding is UTF-8
- Verify the file content format is "name|price|date"

**App Crashes**

- Try restarting the app
- Check if there is enough memory
- If the problem persists, try reinstalling the app

#### Error Messages

- **"No data to export"**：No items have been added to the app
- **"Import failed"**：File format error or corrupted file
- **"Deletion failed"**：Attempting to delete a non-existent item
